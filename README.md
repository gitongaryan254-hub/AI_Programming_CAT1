# Student Info System & Circle Area Scripts

## Overview
This repository contains Python scripts developed as part of coursework exercises. Each script demonstrates practical programming concepts such as input validation, modular design, and mathematical computation.

---

## Files

### 1. `student_info_system.py`
- **Purpose:** Manage student information in a structured way.
- **Features:**
  - Input prompts for student details.
  - Validation to prevent incorrect data entry.
  - Functions for adding, viewing, and updating records.
- **Learning Point:** Reinforces defensive programming and modular design.

### 2. `area_of_circle(radius).py`
- **Purpose:** Calculate the area of a circle given a radius.
- **Features:**
  - Uses the formula πr².
  - Accepts user input and validates numeric values.
- **Learning Point:** Demonstrates how to translate mathematical formulas into executable Python code.
